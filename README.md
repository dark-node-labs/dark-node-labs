# dark-node-labs

Small public tools and experiments for pricing, AI workflows, and web utilities.

## Tools

- [PaycheckDB 2026 Bonus Tax Calculator](https://paycheckdb-2026-calculator.vercel.app/bonus-tax-calculator/) - Free 2026 paycheck and bonus tax calculator with state pages and methodology notes.

- [MSP Pricing Calculator](https://msp-pricing-calculator-vercel.vercel.app/) - free no-email calculator for estimating managed IT services pricing from users, devices, servers, locations, SLA, security add-ons, support labor, tool cost, setup fee, and target margin.
- [Managed IT Services Pricing Calculator](https://msp-pricing-calculator-vercel.vercel.app/managed-it-services-pricing-calculator/) - buyer-side managed IT estimate for small businesses.
- [Managed Services Pricing Calculator](https://msp-pricing-calculator-vercel.vercel.app/managed-services-pricing-calculator/) - broader recurring managed services quote estimator.
- [All Pricing Calculators](https://msp-pricing-calculator-vercel.vercel.app/calculators/) - directory of MSP, managed IT, ROI, IT support, and MSP cost calculators.
